
## 📢 소개
Wetube는 Node.js와 ES6 자바스크립트 기반의 유튜브 클론 코딩 프로젝트입니다. 백엔드는 Node.js 환경에서 구축하였으며, 프론트엔드는 별도의 프레임워크 없이 Vanilla JS와 SCSS만을 활용하여 구현했습니다.
<br><br>

## 💡 구현한 기능

**유저 관리**
* 회원가입, 로그인, 구글 로그인
* 프로필 사진 업로드 및 사용자 정보 수정, 비밀번호 변경

**비디오 관련**
* 비디오 업로드, 수정, 삭제
* 웹캠을 이용한 비디오 녹화 후 업로드
* 비디오 재생 (재생・멈춤, 음량 조절, 풀스크린 모드)

**etc**
* 비디오 검색
* 댓글 작성/수정/삭제
<br><br>
